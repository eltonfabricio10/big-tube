O BigLinux é um sistema operacional brasileiro baseado no Kernel Linux e, atualmente, na distribuição Manjaro Linux, que por sua vez usa o Arch Linux como base. A distro foca em trazer utilitários e uma seleção de programas para ajudar os usuários a fazerem suas tarefas diárias de forma otimizada. A distro usa o ambiente gráfico KDE Plasma e utiliza várias ferramentas próprias desenvolvidas com uma tecnologia chamada "BigBashView" que mescla shell scripts que outras tecnologias geralmente utilizadas em desenvolvimento web para criar aplicativos com design moderno.
